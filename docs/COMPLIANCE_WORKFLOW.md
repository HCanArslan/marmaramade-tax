# Compliance workflow

Open a case for each official question, record the institution/channel/reference, attach private evidence, and move the case through draft, submitted, waiting, response received, clarification, and resolved states. A response does not automatically change legal settings; create a new effective-dated operating profile after review.

The setup checklist covers SGK pension and 4/b questions, artisan eligibility, Etsy seller identity, the exemption bank account, ShipEntegra/ETGB, sales documents, and DDP operations. Audit logs contain metadata only, never document content, addresses, credentials, or tokens.
