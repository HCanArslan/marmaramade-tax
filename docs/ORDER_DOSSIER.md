# Order dossier

An order dossier combines the immutable confirmation snapshot, later adjustments, legal/fee/cost/rate/quote versions, and applicable verified documents. Default evidence includes Etsy order/payment/ledger records, product/package photos, sales/proforma documents, ShipEntegra label/invoice, ETGB, customs calculation, tracking, payout, and material evidence.

Checklist rules may vary by destination, operating mode, incoterm, carrier, return status, exemption status, and confirmation status. Compliance is complete only when every applicable required item is verified. Later costs create adjustments; the original snapshot is never overwritten.
